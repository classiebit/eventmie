## ğŸš€ Installation Guide

### For Fresh Laravel Application

#### Step 1: Create New Laravel Project

```bash
composer create-project --prefer-dist laravel/laravel:^11.0 eventmie-app
cd eventmie-app
```

#### Step 2: Configure Database

Update your `.env` file with database credentials:

```env
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=eventmie_db
DB_USERNAME=your_username
DB_PASSWORD=your_password
```

#### Step 3: Install Eventmie

```bash
composer require classiebit/eventmie
```

#### Step 4: Run Installation Command

```bash
php artisan eventmie:install
```

#### Step 5: Install NPM Dependencies

```bash
npm install
npm run build
```

#### Step 6: Start Your Application

```bash
php artisan serve
```

### For Existing Laravel Application

#### Step 1: Install Eventmie

```bash
composer require classiebit/eventmie
```

#### Step 2: Run Installation Command

```bash
php artisan eventmie:install
```

#### Step 3: Install NPM Dependencies

```bash

```

## ğŸ¨ Configuration

### Route Configuration

Edit `config/eventmie.php` to customize your routes:

```php
'route' => [
    'prefix'       => null,        // Frontend route prefix
    'admin_prefix' => 'admin',     // Admin panel prefix
],
```

### RTL Language Support

Add RTL languages to support right-to-left layouts:

```php
'rtl_langs' => [
    'ar', // Arabic
    'fa', // Persian
    'he', // Hebrew
    'ur', // Urdu
    // Add more as needed
],
```
